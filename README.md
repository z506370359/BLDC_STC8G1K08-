# 🚁 无刷直流(BLDC)风扇电机控制器项目说明书

## 📑 一、 项目简介
本项目是一个基于 **STC8G1K08** 微控制器的无刷直流（BLDC）风扇电机控制器。该项目最初由本能电机的 PIC16F1503 方案移植而来，通过优化软件架构，实现了高效的电机驱动控制。

* **🎯 目标芯片**: STC8G1K08-38I-TSSOP20
* **⏱️ 系统时钟**: 24MHz 内部高精度 IRC
* **🛠️ 编译环境**: Keil C51 (uVision)

---

## ⚙️ 二、 电机参数与核心硬件配置
本项目针对特定的风扇电机进行了参数适配，核心参数如下：

* **🏷️ 电机型号**: D8025 12V
* **🔄 最大转速**: 5000 RPM
* **🧲 传感器类型**: 单霍尔传感器
* **⚡ 换相方式**: 两步换相
* **🔌 驱动拓扑**: 6个MOS管全桥架构（上桥臂由GPIO直接驱动，下桥臂由PCA PWM驱动）

---

## 🔌 三、 硬件引脚分配 (TSSOP20 封装)
以下是微控制器与外围硬件的核心连接定义：

### 1. ⚡ 电机驱动管脚
* **`P1.2` (HU)**: U相上桥臂 MOS 栅极 (推挽输出)
* **`P1.3` (HV)**: V相上桥臂 MOS 栅极 (推挽输出)
* **`P1.1` (LU)**: U相下桥臂 MOS 栅极 (由 CCP0/PCA 输出 PWM)
* **`P1.0` (LV)**: V相下桥臂 MOS 栅极 (由 CCP1/PCA 输出 PWM)
*(注：预留了 P1.6(HW) 和 P1.7(LW) 用于未来扩展三相电机)*

### 2. 📡 传感器与控制输入
* **`P3.2` (INT0)**: 霍尔传感器 A 输入（双沿触发中断）
* **`P1.4` (ADC4)**: 调速指令输入（接收外部 PWM 经过 RC 滤波后的模拟电压）
* **`P3.7` (CMP+)**: 电流检测信号输入（连接采样电阻上的电压）
* **`P3.6` (CMP-)**: 过流保护基准电压输入（连接外部分压器）

### 3. 📊 状态与反馈输出
* **`P1.5` (FG/RD)**: 转速脉冲 / 故障反馈输出
* **`P3.4` (CMPO)**: 比较器输出（预留调试用）
* **`P3.1` (TXD)**: 串口通讯发送端（预留调试用）

---

## 💻 四、 软件控制特性与参数 (调参指南)

本固件内置了完善的状态机与控制算法，以下为核心控制参数：

### 1. 📈 PWM 与 调速设置
* **频率精度**: 使用 PCA 模块生成 8位 PWM，频率约为 **23.4kHz** (FOSC/4/256)。
* **转速查找表**: 支持输入占空比到目标转速的映射。目前默认为 100% 全速运行，如需线性调速可修改 `targetSpeedList` 数组。

### 2. 🚀 软启动控制 (Soft Start)
电机启动时采用软启动策略，防止启动电流过大：
* **起始占空比**: 20%
* **结束占空比**: 30%
* **爬升时间**: 1500 ms

### 3. 🔄 闭环 PI 控制
系统内置 PI (比例-积分) 控制器以稳定转速：
* **控制周期 (PID_CYCLE)**: 50 ms
* **比例增益 (P)**: 12 (增大可加快响应，但过大易振荡)
* **积分增益 (I)**: 3 (用于消除稳态误差，过大易超调)

### 4. 📐 换相与超前角调整
* **换相延迟 (RISING_DELAY)**: 设定为 400 个定时器1计数（约 **200μs**）。增大该值可以在电机高速运行时获得更大的超前角补偿。

---

## 🚨 五、 硬件保护与安全机制

### 1. ⚡ 硬件过流保护 (CMP 模块)
* **工作原理**: 将电流采样电阻的电压送至 `P3.7 (CMP+)`，与外部电阻分压提供的基准电压 `P3.6 (CMP-)` 进行比较。
* **触发动作**: 当电流超过设定阈值时，触发中断，软件会在 2~5μs 内立即关闭所有 MOS 管，防止烧毁。

### 2. 🛑 堵转检测与重启
* **检测机制**: 若定时器1溢出（即超过 **32.768ms** 没有发生霍尔换相），系统判定为电机堵转。
* **重启策略**: 检测到堵转后，电机会进入等待状态，延迟 **5000ms** (5秒) 后尝试重新启动。

---

## 🛠️ 六、 外围电路设计建议
* **MOS驱动**: 所有的 MOS 栅极控制引脚均配置为推挽输出，建议 MCU 与 MOS 管之间增加栅极驱动 IC（如 IR2101）以保证驱动能力。
* **调速滤波**: P1.4 的调速输入端需外加 RC 滤波电路，推荐参数为 $R = 10k\Omega$, $C = 100nF$（时间常数 1ms）。
* **电源去耦**: VCC 引脚旁务必放置 100nF 的去耦电容。
* **过流基准设计**: 若系统 VCC 为 5V，可使用 10kΩ(接VCC) 和 2.2kΩ(接GND) 串联分压，在 `P3.6` 处获得约 0.9V 的比较器阈值电压。
