================================================================================
 无刷直流风扇电机控制器 - STC8G1K08 移植版
 原始项目: PIC16F1503 Fan-Single-HallSensor.X (本能电机)
================================================================================

目标芯片:   STC8G1K08-38I-TSSOP20
系统时钟:   24MHz 内部高精度IRC
编译工具:   Keil C51 (uVision)
电机型号:   D8025 12V 5000RPM, 单霍尔传感器, 两步换相


1. 工程文件说明
-----------------
  stc8g.h         寄存器定义 (仅包含本项目用到的SFR)
  bldc_config.h   电机参数、引脚映射、宏定义
  main.c          完整固件 (初始化、中断、状态机、PI控制器)


2. Keil工程创建步骤
---------------------
  1) 新建工程 → 器件选择 "Generic 8051" 或 STC 器件库(如已安装)
  2) 添加 main.c 到源文件组
  3) 工程选项:
     - 晶振频率 Xtal = 24.000 MHz
     - 存储器模型 Memory Model: Small
     - 代码ROM大小 Code ROM Size: Large (8KB)
     - 片外代码/数据: 无
  4) 输出设置 → 勾选 "生成HEX文件"
  5) 编译 (F7)
  6) 通过 STC-ISP 工具下载:
     - 选择型号 STC8G1K08
     - IRC频率设为 24MHz
     - P5.4 设为 I/O口 (除非需要外部复位)
     - 点击"下载/编程"后给芯片重新上电


3. 引脚分配 (TSSOP20)
-----------------------
  引脚  端口    方向    功能说明
  ----  ------  ------  ----------------------------------
   1    P1.2    输出    HU - U相上桥臂MOS栅极 (GPIO)
   2    P1.3    输出    HV - V相上桥臂MOS栅极 (GPIO)
   3    P1.4    输入    ADC4 - 调速指令 (PWM经RC滤波后的电压)
   4    P1.5    输出    FG/RD - 转速脉冲/故障反馈
   5    P1.6    输出    HW - W相上桥 (预留, 扩展三相用)
   6    P1.7    输出    LW - W相下桥 (预留, 扩展三相用)
   7    P5.4    -       复位 / 空
   8    VCC     电源    3.3V 或 5V
   9    P5.5    -       空
  10    GND     地      -
  11    P3.0    -       空 (可用作INT4)
  12    P3.1    输出    串口TX (调试用, 可选)
  13    P3.2    输入    INT0 - 霍尔传感器A (双沿中断)
  14    P3.3    -       空
  15    P3.4    输出    CMPO - 比较器输出 (调试用, 可选)
  16    P3.5    -       空
  17    P3.6    输入    CMP- 过流保护基准电压 (外部分压)
  18    P3.7    输入    CMP+ 电流检测信号 (采样电阻电压)
  19    P1.0    输出    CCP1 → LV - V相下桥臂MOS栅极 (PCA PWM)
  20    P1.1    输出    CCP0 → LU - U相下桥臂MOS栅极 (PCA PWM)


4. 与PIC版的关键差异对照
--------------------------
  PIC16F1503                           STC8G1K08
  ──────────────────────────────────── ────────────────────────────────────
  PWM1 (25kHz, 10位精度)               PCA 8位PWM (~23.4kHz, FOSC/4/256)
  CWG互补输出 (硬件死区)               两个PCA通道, 通过CCAPM开关切换
  CLC硬件与门 (自动逻辑路由)           纯软件: INT0中断中AllMosOff()
  DAC+CMP2 → CWG自动关断              内置比较器 → 软件中断关MOS
  IOC双沿 (RA3)                        INT0双沿 (IT0=0, STC特有)
  TMR1 1μs/计数, 65ms溢出              定时器1 12T: 0.5μs/计数, 32ms溢出
  TMR2中断实现超前角                    PCA溢出中断轮询 (~23.4kHz)
  4MHz有效指令速度 (1MIPS/MHz)         24MHz 1T模式 (24 MIPS)


5. 过流保护电路说明
---------------------
  硬件连接:
    - 电流采样电阻 → 电压送到 P3.7 (CMP+)
    - 基准电压分压器 → 电压送到 P3.6 (CMP-)
    - 当采样电压 > 基准电压时: 比较器触发中断
    - 中断服务程序立即关闭全部MOS管
  
  可选方案: 使用内部1.19V基准代替P3.6外部分压
    修改 CMPCR1 从 0xA4 改为 0xA0 (NIS=0选择内部基准)

  注意: PIC版使用CWG硬件自动关断, 响应时间<1μs;
        STC版使用软件中断关断, 响应延迟约2~5μs
        风扇电机应用中这个延迟是可以接受的
        如需更快响应, 可在MOS驱动电路上另加硬件比较器


6. 调参指南
-------------
  软启动参数:
    - SOFT_START_START_DUTY: 起始占空比% (默认20)
    - SOFT_START_END_DUTY:   结束占空比% (默认30)
    - SOFT_START_DELAY:      爬升时间ms (默认1500)
    
  PI控制器:
    - PWM_P_PARAMETER: 比例增益 (默认12)
    - PWM_I_PARAMETER: 积分增益 (默认3)
    - PID_CYCLE:       控制周期ms (默认50)
    - 增大P: 响应加快, 但可能振荡
    - 增大I: 消除稳态误差, 但可能超调
    - 减小P/I: 响应变慢但更稳定
    
  转速查找表:
    - targetSpeedList[]: 输入占空比% → 目标转速%
    - 目前全部填100 (全速运行), 修改可实现变速
    - 示例(线性比例):
        targetSpeedList[0]=0, [10]=10, [50]=50, [100]=100

  超前角:
    - RISING_DELAY: 换相延迟, 定时器1计数值 (0.5μs/计数)
    - 默认400 = 200μs, 增大可在高速时获得更大超前角
    
  堵转检测:
    - 定时器1溢出32.768ms = 判定堵转
    - RESTART_DELAY: 堵转后等待时间 (默认5000ms)


7. 外围电路设计要点
---------------------
  - 所有MOS栅极驱动引脚均为推挽输出
  - MCU与MOS管之间需要栅极驱动IC (如IR2101等)
  - P1.4调速输入需要RC滤波:
      R = 10kΩ, C = 100nF (时间常数τ=1ms)
  - VCC引脚旁放100nF去耦电容
  - 霍尔传感器输出接P3.2, 根据传感器类型加上/下拉电阻
  - 电流采样电阻: 典型值 10~50mΩ
  - CMP-基准分压器示例:
      VCC=5V, R1=10kΩ接VCC, R2=2.2kΩ接GND → 约0.9V阈值


8. STC-ISP 下载设置
---------------------
  - 芯片型号: STC8G1K08
  - IRC频率: 24.000 MHz
  - 低压复位: 根据需要选择 (如2.4V)
  - P5.4功能: 设为I/O口 (除非用作复位)
  - 硬件连接: P3.0(RXD)和P3.1(TXD)接USB转串口模块
  - 操作: 在STC-ISP中点击"下载/编程"后, 给芯片断电再上电
